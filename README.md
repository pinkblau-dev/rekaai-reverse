## Installation
1. Clone Repository
```bash
git clone https://github.com/pinkblau-dev/rekaai-reverse.git
```
2. Setup Envoirement
```bash
cp .env.example .env
```

3. Install Requirements
```bash
pip install -r requirements.txt
```

4. Run Script
```bash
main.py
```
